🍽️ Amity Campus Eats

Amity Campus Eats is a web-based online canteen ordering system for Amity University. It helps students and faculty order food easily through a user-friendly interface.

🔧 Features

View menu with prices
Add items to cart
Mobile responsive UI
Secure online payments
Final success message after order

🧑‍💻 Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
